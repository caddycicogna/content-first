Facts about the Font

In 1955 Courier was created by a man named Howard "Bud” Kettler. Courier is classified as a monospaced slab serif type face.

A who’s-a-what? We’ll break it down.

Monospaced refers to the fact that every letter and character in this font each occupy the same amount of horizontal space. Slab serif refers to a font that can be characterized by its thick, block like serifs found at the ends of letters or characters within the font.

Courier was originally a commissioned project for use in typewriters at IBM in the 1950’s. It quickly became the standard font for the typewriter industry, but IBM did fail to take initiative ownership over it before different versons were being made of this font for typewriters.
